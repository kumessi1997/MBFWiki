## Summary

Official akaBot license price list (Vietnamese, for the Vietnamese market). Annual license pricing per component.

## License pricing (VNĐ, annual)

| Product | List price (VNĐ/year) | Notes |
|---|---|---|
| akaBot Studio | 47,000,000 | Design tool; **free during implementation** if using FPT deployment services |
| akaBot Attended Agent | 35,000,000 | Human-triggered bot; 1 license = 1 machine |
| akaBot Unattended Agent | 58,750,000 | Scheduled bot controlled by Center; 1 license = 1 machine |
| akaBot Center | 141,000,000 | Orchestration server; 1 license = 1 machine/server |

## USD equivalents (approximate, at ~25,000 VNĐ/USD)

| Product | ~USD/year |
|---|---|
| Studio | ~$1,880 |
| Attended Agent | ~$1,400 |
| Unattended Agent | ~$2,350 |
| Center | ~$5,640 |

## Support SLA (included with license)

- Channels: email (support@akabot.com), support website, community portal, docs
- Hours: Mon–Fri, 9am–6pm GMT+7 (excluding Vietnamese public holidays)
- Response times: High urgency < 2 hrs response / < 8 hrs resolution; Medium 8–16 hrs; Low 16–24 hrs

## Key observations

- Studio is waived during implementation — lowers entry cost and encourages FPT-led deployments
- Center pricing (~$5,640/yr) is significantly cheaper than UiPath Orchestrator or Automation Anywhere Control Room, which typically cost $10,000–$30,000+/year
- Pricing is per-machine, not per-process — encourages scaling via adding machines/bots

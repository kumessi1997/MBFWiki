## Overview

**GeoBase Cloud** is FPT IS's **Geographic Information System (GIS) Platform** — a cloud-native solution for managing, processing, sharing, and visualizing geospatial data. It solves fragmented spatial data workflows across field engineers, government agencies, and enterprises by converting diverse GIS formats into centralized, accessible web map services.

Primary target buyers: government departments (Natural Resources, Construction, Transport), project management boards, construction contractors, survey/cadastral engineers, and agricultural enterprises needing spatial data management.

## Core capabilities

**8 key functions:**
1. Multi-format data ingestion → DWG, DGN, SHP, GML, TAB, Excel, GeoJSON, GeoPackage, GeoTIFF, GDB, LAS, OBJ → converted to map layers
2. Repository-based data management with role-based access and sharing permissions
3. Map visualization on multiple base maps (traffic, terrain, satellite imagery)
4. Web map editing, publishing, spatial analysis, statistical reporting
5. Mobile data collection and measurement (Android + iOS)
6. Multi-national coordinate system support: VN-2000, WGS-84, HN72, UTM, Lambert-93, Greek Grid, British National Grid, LV95, and more
7. 3D data, BIM, point cloud, remote sensing, IoT integration → **Digital Twin** construction
8. Centralized data environment for project management and construction operations

**Key features:**
- Quickly converts DGN/DWG/GML/TAB files to web-GIS services, preserving original presentation styles
- Thematic map editing and creation from managed data layers
- Flexible database schema (Water resources, Minerals, Environment, Sea & Islands, Construction, Traffic, Dyke management, etc.)
- Document management (centralized file storage)
- Data approval workflows with digital signature integration (reduces printing costs)
- Open platform adhering to OGC standards → rapid development of specialized applications on top

**Geospatial Data Viewer module:**
- Web-based 3D model and point cloud visualization (no heavy software installation needed)
- User location positioning on drawings and 3D models
- Measurement tools with PDF/Excel report auto-generation
- SSO integration (LDAP, Azure AD, Keycloak); deployed via Docker or Kubernetes

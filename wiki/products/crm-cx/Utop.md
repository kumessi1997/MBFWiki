## Overview

**Utop** is a member company of [[FPT Corporation]], backed jointly by **FPT and SBI Holdings**. Website: utop.io. Utop specializes in AI-embedded CRM, loyalty marketing, and customer data platforms, operating under the brand umbrella **FPT CX Suite**. It is distinct from [[FPT IS]] (which distributes Utop products as part of its CRM & Customer Experience category) — Utop is the product developer.

FPT CX Suite is positioned as a Southeast Asia-localized enterprise platform competing directly with Salesforce, Microsoft Dynamics 365, Oracle, and Comarch. It holds **ISO 27001** and **SOC 2 Type II** certifications and is available on the **Microsoft Azure Marketplace**.

## Platform: FPT CX Suite

**FPT CX Suite** is the branded umbrella for all Utop products. It is built on a microservices, API-first architecture with on-premises and SaaS deployment options.

### Core modules

| Product | Description |
|---|---|
| **LoyaltyOS** | End-to-end loyalty marketing platform; blockchain-powered points currency; omni-channel; 4–6 month deployment |
| **UEngage** | Customer Data Platform (CDP) + marketing automation; unified customer profile, segmentation, campaign execution |
| **UReach** | AI CRM Agent using **Large Action Model (LAM)**; adaptive outreach campaigns; predictive churn and next-best-offer |
| **UAgent** | AI Agent Builder / CRM Copilot; low-code agent creation for sales and service workflows |
| **Promotion Engine** | Configurable promotion and voucher management across channels |
| **UCDP** | Unified Customer Data Platform layer underpinning all modules |
| **CRM Next** | Core CRM system for sales pipeline and customer lifecycle management |
| **Saleswork AI** | AI-powered sales force automation app |
| **UPOS** | Point-of-sale system integrated with loyalty and CRM |
| **Headless Commerce** | CMS, POS, Checkout OS for omni-channel commerce |

### Utop atWork (B2B2E)

**Utop atWork** is an enterprise extension of Utop's loyalty ecosystem targeting HR departments. Employees earn and spend Utop points for canteen, shopping, entertainment, and partner rewards — using the same points currency as consumer loyalty. Digitizes employee benefit administration.

## Competitive positioning

FPT CX Suite is differentiated from global CRM/loyalty vendors by:
- **Faster deployment**: 4–6 months vs. 12–18 months
- **Lower TCO**: vs. Salesforce and MS Dynamics 365
- **SEA localization**: Vietnamese regulatory environment, local payment integration, local language support
- **AI-native**: UReach uses LAM (not just LLM chatbot) for genuine agentic outreach
- **Blockchain loyalty currency**: unique technical differentiator in LoyaltyOS

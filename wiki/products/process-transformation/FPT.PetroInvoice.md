## Overview

**FPT.PetroInvoice** is a specialized **e-invoice SaaS solution for Vietnam's petroleum/fuel retail sector**, developed by **FPT IS**. It enables petrol stations to issue e-invoices per transaction in compliance with Government Directive 1284/CĐ-TTg (mandatory from 12/2023), using a software-first approach that eliminates the need for expensive hardware upgrades.

Operates under the [[FPT.eInvoice]] umbrella (same contact: FPT.eInvoice@fpt.com; same website: einvoice.fpt.com.vn). It is a **vertical specialization** of the FPT.eInvoice ecosystem for the fuel/petroleum sector, not a fully independent brand.

**Market driver:** Directive 1284/CĐ-TTg requires all petrol stations to issue e-invoices per retail fuel transaction from 12/2023. Traditional compliance costs: **400M–1B VND per station** for hardware + software upgrades per pump. FPT's solution: from **2 triệu VND/station** + **50đ/invoice**.

## Two Solutions

### Solution 1 — FPT.PetroInvoice AI
**For:** All fuel pump types (mọi loại trụ bơm)
**Deploy time:** 1 day per petrol station

**How it works:**
- Smart Camera reads the transaction amount displayed on the pump's electronic screen
- Data sent to FPT.PetroInvoice system → auto-issues e-invoice per GDT regulation
- Covers both invoice-requesting customers and non-requesting customers (regulatory requirement)

**Features:**
- Integrates with existing petroleum management software for daily data sync and audit
- QR Code dynamic payment support for customers on mobile (TPBank, NAPAS, VNPAY)
- Auto-alerts for abnormal invoice status via email/SMS
- Already deployed in Nam Dinh and Hai Duong provinces

### Solution 2 — Electronic Display Unit Replacement
**For:** Pumps without built-in data transmission capability
**Deploy time:** 1–2 days per petrol station

**How it works:**
- Replaces old electronic display with a new smart unit
- New unit transmits fuel transaction data to the station's centralized data collection system
- Eliminates need for full pump replacement

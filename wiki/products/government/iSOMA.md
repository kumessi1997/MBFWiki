## Overview

**iSOMA** is [[FPT IS]]'s comprehensive document digitization solution, operated by the IDA (Innovation & Digital Assets) division. It converts physical documents — paper, images, audio — into structured digital data using a combination of AI-OCR automation, trained human operators, and specialized scanning equipment.

Tagline: *"Số hoá toàn diện — Hiệu suất · Nhanh chóng · Bảo mật"* (Comprehensive Digitization — Efficiency · Speed · Security)

iSOMA is positioned as a **service**, not just software: FPT provides the full stack of people, devices, technology, and physical workspace (on-site or at FPT's ODC rooms).

## Target markets

**Government:** Central and local ministries — construction, natural resources & environment, home affairs, archives, civil records, administrative procedures

**Enterprise:** Banking, insurance, retail, factories, hospitals, construction/infrastructure

## 8-step digitization process

1. Collect documents (image, PDF, paper)
2. Classify documents
3. Scan (auto scanner / flatbed / book scanner A4–A0 / specialized)
4. Check document quality
5. OCR & Data entry (AI-OCR for good-quality docs; manual for low-quality)
6. Check data entry
7. Export & store
8. Search & retrieve *(integrated with specialized archiving software)*

## Technology architecture

**3-tier web platform:**
- Presentation layer: iSOMA web (HTTPS for Customer / Data Entry / Admin roles)
- Application layer: Web-services → Customer mgmt, Input mgmt, OCR & Data entry, Output mgmt, Dashboard/Reports, User/permission mgmt, Data sync/sharing, System admin
- Data layer: File storage system + MariaDB + Elasticsearch index

**External integrations:** Digital Signature, AI-OCR engine, external CSDL/systems

## Smart Data Digitization model

Uses AI-OCR to scan, digitize, and auto-classify documents into attribute-based folder structures. Benefits:
- Reduce raw data management costs
- Auto-arrange digitized info into attribute folders
- High-accuracy auto-recognition via continuous ML learning
- Secure version management; easy retrieval anytime, anywhere

## Relationship to ArchiveNex

iSOMA and [[ArchiveNex]] form FPT IS's full document lifecycle:
- **iSOMA** = digitize (scan → OCR → data entry → structured digital data)
- **ArchiveNex** = archive & retrieve (long-term storage, workflow, e-signature, datalake, big data analytics)

iSOMA exports metadata in JSON, XML, TXT, CSV formats ready for ArchiveNex ingestion.
